Name decadal_fullfield
Long Name 1.1 Fullfield initialisation ensembles
Type Initial Condition
Description The ensemble is produced by SST perturbation (i.e. different inputs). The first element in each ensemble is initialised by the fullfield- assimilation run. The other nine members in each ensemble use identical atmospheric initial conditions. The oceanic initial conditions are also the same except that the SSTs are perturbed with white noise in the order of 5x10-7 K. The SST perturbation is done via an external script and entered through the oceanic startup dump file. See Smith, D. M., S. Cusack, A. W. Colman, C. K. Folland, G. R. Harris and J. M. Murphy, 2007, " Improved surface temperature prediction for the coming decade from a global climate model", Science, 317, 796-799, doi:10.1126/science.1139540
